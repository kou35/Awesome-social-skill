# awesome-social-skills

中文 | [English](README_EN.md)

> 这是一个精心整理的人物与关系类 AI skill 集合，聚焦职场、亲密关系、赛博名人、名人与其他有趣场景。  
> 本仓库收录那些把记忆、性格、习惯、相处方式与经验沉淀为可复用 AI skill / persona / agent 的项目、模板与灵感。  
> 无论是实用的、好玩的，还是抽象但有意思的 skill，都欢迎在这里被发现、收集与探索。

<a id="why"></a>

## 为什么有这个仓库 🤔

- 现在各种人物、关系类的 skill 层出不穷，但分散在不同仓库和社区里。
- 这个仓库希望把它们整理成一个更容易浏览、补充和探索的索引。
- 不管你是想找灵感、找乐子，还是研究 persona / skill / agent 的表达方式，都可以从这里开始。
